# Расписание занятий

Это Android приложение для просмотра расписания занятий и быстрого просмотра текущей успеваемости.

## Фишки приложения
- Просмотр расписания на любой день
- Виджет на главный экран
- Баллы из lk и edu всегда под рукой
- open-source

## Сборка
1. Клонируйте репозиторий
2. Откройте проект в Android Studio
3. Соберите APK
